# 磨损体积计算公式（修正版）

## 1. 原始公式 (1-30号刀具)

### 正向：半径 → 体积

```
V = 2π·ΔR·N·[TR - (T/2)·ΔR + R·tan(θ)·ΔR - (2/3)·tan(θ)·(ΔR)²]
```

其中：
- V: 磨损体积 (mm³)
- ΔR: 磨损半径 (mm)
- T: 刀具厚度 (mm), 默认 25.4
- R: 刀具位置半径 (mm), 默认 241.3
- θ: 刀具角度 (度), 默认 20°
- N: 刀具数量, 默认 1

### 反向：体积 → 半径

求解三次方程，选择最小的正实根作为物理解。

---

## 2. 新公式 (31-44号刀具)

### 参数定义

- T = 25.4 mm
- H = 2413 mm (刀具高度)
- θ = 20°
- **d = (i-30) × (70/13)** (度)，其中 **i 为刀具编号**

### 中间变量计算

#### 计算 h₀:

```
h₀ = [(T/2 + R/tan(θ)) · tan(90°-θ)] / [1 - tan(d)]
```

#### 计算 h₁:

```
h₁ = [R/tan(d) - T/2] · [1 - (sin(θ)·cos(θ))/sin(90°+θ-d)] · tan(d)
```

### 判定条件

当 **h₀ = 2R** 时，计算判定值：

```
condition = h₀/tan(d) - h₀/tan(θ+d)
```

**判定规则**：
- 若 **condition > 0**，使用**公式1**
- 否则，使用**公式2**

### 公式1：V = V₀ - V₁

```
V₀ = π · [R/tan(d) + T/2] · h₀ · [H - h₀/3]

V₁ = π · [R/tan(d) - T/2] · h₁ · [H - h₁/3]

V = V₀ - V₁
```

### 公式2：

```
V = π · condition · h₀ · [H - h₀/3]
```

**注意**：公式2不是1-30号刀具的公式，而是基于 condition 值的新公式。

### 反向：体积 → 半径

使用数值方法求解（scipy.optimize.fsolve 和 brentq）。

## 3. 特殊规则

### 44号刀具
44号刀具的磨损量与43号刀具完全一致：

```
V₄₄ = V₄₃
d₄₄ = d₄₃ = (43-30) × 70/13 = 70°
```

## 8. 关键修正点

与之前版本相比的主要修正：

1. ✓ **d 的计算**：从固定值 `(7-30)×70/13` 改为 `(i-30)×70/13`，其中 i 是刀具编号
2. ✓ **h₀ 公式**：使用 `tan(90°-θ)` 而不是 `tan(2θ-d)`
3. ✓ **h₁ 公式**：
   - 分子使用 `R/tan(d)` 而不是 `R/tan(θ)`
   - 角度使用 `90°+θ-d` 而不是 `2θ-d`
   - 最后乘以 `tan(d)` 而不是加 `tan(θ)`
4. ✓ **V₀ 和 V₁**：使用 `R/tan(d)` 而不是 `R/tan(θ)`
5. ✓ **判定条件**：`h₀/tan(d) - h₀/tan(θ+d) > 0`
6. ✓ **公式2**：使用 `π·condition·h₀·[H-h₀/3]` 而不是1-30号刀具的公式
