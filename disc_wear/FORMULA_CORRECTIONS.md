# 刀具磨损体积计算公式修正记录

## 修正日期
2026-01-23

## 修正内容

### 1. h₀公式修正

#### 错误版本
```
h₀ = [(T/2 + R/tan(θ)) · tan(90°-θ)] / [1 - tan(d)]
```

#### 正确版本
```
h₀ = [(T/2 + R/tan(d)) · tan(90°-θ)] / [tan(90°-θ) - tan(d)]
```

#### 修正说明
- **分子**: `R/tan(θ)` 改为 `R/tan(d)` - 应该使用角度d而不是θ
- **分母**: `1 - tan(d)` 改为 `tan(90°-θ) - tan(d)` - 避免tan(d)接近1时的奇异性

#### 影响
- 避免了当tan(d)≈1（d≈45°）时的数值不稳定
- 正确反映了角度d在公式中的作用
- 刀具31-37号的计算结果更加合理

---

### 2. h₁公式修正

#### 错误版本
```
h₁ = [R/tan(d) - T/2] · [1 - (sin(θ)·cos(θ))/sin(90°+θ-d)] · tan(d)
```

#### 正确版本
```
h₁ = [R/tan(d) - T/2] · [1 - (sin(d)·cos(θ))/sin(90°+θ-d)] · tan(d)
```

#### 修正说明
- **括号内**: `sin(θ)·cos(θ)` 改为 `sin(d)·cos(θ)` - 应该使用sin(d)而不是sin(θ)

#### 影响
- 正确反映了角度d对h₁的影响
- 改善了所有刀具的计算精度

---

### 3. 条件判断修正

#### 错误版本
```
if condition > 0:
    使用公式1
else:
    使用公式2
```

#### 正确版本
```
if condition > T:
    使用公式1
else:
    使用公式2
```

其中 `condition = 2R/tan(d) - 2R/tan(θ+d)`

#### 修正说明
- 判断阈值从0改为T（25.4mm）
- 更符合物理意义

#### 影响
- 改善了公式切换的合理性
- 减少了边界情况下的异常值

---

### 4. 边界保护添加

#### 问题
当 d = 70° 时，tan(90°-θ) = tan(70°)，导致分母为0

#### 解决方案
```python
epsilon = 1e-3  # 阈值
if |tan(90°-θ) - tan(d)| < epsilon:
    h₀ = 2R  # 使用近似值
else:
    h₀ = [(T/2 + R/tan(d)) · tan(90°-θ)] / [tan(90°-θ) - tan(d)]
```

#### 影响
- 解决了刀具43号（d=70°）的奇异点问题
- 避免了数值溢出和负无穷大的情况
